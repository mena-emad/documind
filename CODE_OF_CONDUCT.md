# Contributor Covenant Code of Conduct

## Our Pledge

We as contributors and maintainers of DocuMind pledge to make participation in this project a welcoming, respectful, and inclusive experience for everyone.

We commit to:

- Being considerate and courteous in all communication.
- Showing empathy and respect for differing viewpoints and experiences.
- Giving constructive feedback that helps contributors learn and grow.
- Welcoming newcomers and helping them succeed.

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Using welcoming and inclusive language.
- Being respectful in discussions and code reviews.
- Accepting constructive criticism gracefully.
- Focusing on the problem, not the person.

Examples of unacceptable behavior by participants include:

- Harassment, discrimination, or exclusionary language.
- Derogatory remarks, insults, or personal attacks.
- Publishing private information without permission.
- Disruptive behavior in project spaces.

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues and pull requests
- Commit messages and code review comments
- Project discussions, chats, and community channels

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

If you experience or witness unacceptable behavior, please report it to the project maintainers.
